# Quinton Larson Portfolio

This folder is a static portfolio site. It can be hosted on GitHub Pages, Netlify, Vercel, or any static web host.

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any local static server.

## GitHub Pages

1. Create a GitHub repository for the portfolio.
2. Upload the contents of this folder.
3. In the repository settings, enable GitHub Pages from the main branch.
4. Optional: connect a custom domain after the GitHub Pages URL is working.

## Included

- `index.html`
- `styles.css`
- `script.js`
- `assets/` with portfolio media and resume
